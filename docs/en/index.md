# Statistics Canada OrchardCore Documentation

Welcome to the `StatCan.OrchardCore` documentation. Learn how to use our collection of application and SaaS modules and components, and Themes for building your project.

## OrchardCore CMS

The project is built on OrchardCore CMS version `1.0.0-rc2-16092`. To learn more about the underlying framework, take a look through the [Orchard Core CMS documentation](https://docs.orchardcore.net) and the [Orchard Core Github repository](https://github.com/OrchardCMS/OrchardCore).
